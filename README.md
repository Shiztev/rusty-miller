# rusty-miller
Rust implementation of the Miller-Rabin primality test.

# Requirements
Rust >= 1.60.0



# Commands
First, install the dependancies: `cargo install --path .`

To run: `cargo run -- <bits> <count=1> <k=10>`

To build then run: `cargo build`

rusty-miller <bits> <count=1> <k=10>
